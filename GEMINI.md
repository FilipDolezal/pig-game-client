# Gemini Project: sp-client

## Project Overview

This project is the client-side implementation of a client-server application for playing the game "Pig". The client is a Java Swing application.

The server-side implementation is located in a separate project. The requirements for the client are located in the `docs` directory.

The project is structured into three main packages:
- `view`: Contains the Swing UI components.
- `controller`: Contains the logic for controlling the UI and handling user input.
- `net`: Contains the networking code for communicating with the server.

## Building and Running

This project does not use a build management tool like Maven or Gradle. It can be built and run using an IDE like IntelliJ IDEA.

### From IntelliJ IDEA

1.  Open the project in IntelliJ IDEA.
2.  The IDE should automatically detect the project structure.
3.  The project includes a run configuration for the `Main` class.
4.  Build and run the project.

## Development Conventions

The project follows standard Java conventions. The source code is located in `src`.

The application uses the Swing framework for the graphical user interface.
